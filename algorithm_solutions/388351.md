# [유연근무제](https://school.programmers.co.kr/learn/courses/30/lessons/388351)

### Solution

각 사람 i 별로 설정 시간 + 10 을 넘겼는지 확인

```
int solution(vector<int> schedules, vector<vector<int>> timelogs, int startday) {
    int answer = 0;
    
    for(int i = 0 ; i < schedules.size() ; i++)
    {
        int m = schedules[i]/100*60 + schedules[i]%100 + 10;
        int day = startday;
        bool able = true;
        for(int j = 0 ; j < timelogs[i].size() && able ; j++)
        {
            if((day-1)%7 < 5)
            {
                int cTime = timelogs[i][j]/100*60 + timelogs[i][j] % 100;
                if(cTime > m)able = false;
            }
            day++;
        }
        if(able) answer++;
    }
    
    return answer;
}
```
