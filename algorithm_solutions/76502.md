# [괄호회전](https://school.programmers.co.kr/learn/courses/30/lessons/76502)

Stack을 사용해 괄호가 정상적인지 검사
O(n^2)

### Solution

```
int solution(string s) {
    int answer = 0;
    stack<char> st;
    for (int idx = 0; idx < s.size(); idx++)
    {
        st = stack<char>();
        bool able = true;
        for (int i = 0; i < s.size(); i++)
        {
            char c = s[(idx + i) % s.size()];
            if (c == '}') {
                if (st.empty() || st.top() != '{')
                {
                    able = false;
                    break;
                }
                else {
                    st.pop();
                }
            }
            else if (c == ']') {
                if (st.empty() || st.top() != '[')
                {
                    able = false;
                    break;
                }
                else {
                    st.pop();
                }

            }
            else if (c == ')') {
                if (st.empty() || st.top() != '(')
                {
                    able = false;
                    break;
                }
                else {
                    st.pop();
                }

            }
            else st.push(c);
        }
        if (able && st.empty())
        {
            answer++;
        }
    }

    return answer;
}
```
