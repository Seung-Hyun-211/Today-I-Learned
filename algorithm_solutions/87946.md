# [피로도](https://school.programmers.co.kr/learn/courses/30/lessons/87946)

### Solution
완전탐색, 재귀<br>
재귀함수의 depth가 8을 넘지 않음

```
int solution(int k, vector<vector<int>> dungeons) {
    int answer = 0;
    for (int i = 0; i < dungeons.size(); i++)
    {
        //탐색 가능?
        if (k >= dungeons[i][0])
        {
            vector<int> temp(2);
            temp[0] = dungeons[i][0];
            temp[1] = dungeons[i][1];

            //현재 위치를 탐색한다고 가정하고 더 좋은 조건을 찾는다.
            dungeons.erase(dungeons.begin() + i);
            answer = max(answer, 1 + solution(k - temp[1], dungeons));
            dungeons.insert(dungeons.begin() + i, temp);
        }
    }

    return answer;
}
```
