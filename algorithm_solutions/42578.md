### Solution

1. 잘못된(어려운) 풀이<br>
   예 ) 3종류의 의류가 있을 때 3종류 중 1개만 고르는 경우 + 3종류 중 2개만 고르는 경우 + 3종류 중 3개를 고르는 경우 

#### 간단한 풀이
각 의류에 "입지 않음"의 경우를 추가하고<br>
마지막에 "모두 입지 않음"의 경우를 뺀다.

```
int solution(vector<vector<string>> clothes) {
    int answer = 1;
    map<string, int> m;

    for (auto iter = clothes.begin(); iter != clothes.end(); iter++)
        m[(*iter)[1]]++;

    for (auto iter = m.begin(); iter != m.end(); iter++)
        answer *= (iter->second + 1);

    return answer-1;
}
```
