# [삼각 달팽이](https://school.programmers.co.kr/learn/courses/30/lessons/68645)

### Solution

아래, 오른쪽, 위 순서대로 n, n-1, n-2 ... 만큼씩 이동하며 숫자를 저장

```
vector<int> solution(int n)
{
    vector<vector<int>> arr(n, vector<int>(n));

    //down, right , up
    int dirx[3] = { 0,1,-1 };
    int diry[3] = { 1,0,-1 };

    int dir = 0;
    int x = 0;
    int y = -1;
    int idx = 1;

    for (int i = n; i > 0; i--)
    {
        for (int j = 0; j < i; j++)
        {
            x += dirx[dir];
            y += diry[dir];
            arr[y][x] = idx++;
        }
        dir = (++dir) % 3;
    }
    vector<int> answer(idx-1);
    idx = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j <= i; j++)
        {
            answer[idx++] = arr[i][j];
        }

    return answer;
}
```
