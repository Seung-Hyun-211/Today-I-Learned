# [연속 부분 수열 합의 개수](https://school.programmers.co.kr/learn/courses/30/lessons/131701)

### Solution

부분 수열의 합 set : 겹칠 수 있으니 set 사용<br>
a ~ b 까지의 합 DP : col -> 수열 길이 / raw -> 현위치 까지의 합

```
int solution(vector<int> elements) {
    int answer = 0;
    vector<vector<int>> dp(1000, vector<int>(1000));
    set<int> ans;
    for (int i = 0; i < elements.size(); i++)
    {
        dp[0][i] = elements[i];
        ans.insert(elements[i]);
    }
    int len = elements.size();
    for (int i = 1; i < len; i++)
    {
        for (int j = 0; j < len; j++)
        {
            dp[i][j] = dp[i - 1][(len + j - 1) % len] + elements[j];
            ans.insert(dp[i][j]);
        }
    }

    return ans.size();
}
```
